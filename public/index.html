PROJECT OVERVIEW

BMW DriveIQ is a real-time driver behaviour intelligence system that classifies driving style from raw vehicle telemetry using machine learning, then generates personalised coaching feedback powered by a large language model.

The system replicates the AI layer found inside BMW ConnectedDrive / iDrive 9 — turning sensor streams into actionable, human-readable intelligence that helps drivers understand habits, manage risk, and improve technique.
HOW IT WORKS

1 — Telemetry Simulation
Physics-informed simulator generates realistic multi-channel vehicle telemetry for five driver archetypes.

2 — Feature Engineering
Raw telemetry → 10-second sliding windows → 52 statistical features extracted per window.

3 — XGBoost Classification
Gradient-boosted classifier trained on 11,600 labelled windows. 99.5% test accuracy.

4 — LLM Coaching Layer
Classification + stats → Groq-hosted LLaMA 3.3 70B → personalised BMW-style coaching report.

DRIVER PROFILES

● SMOOTH
Controlled inputs. Low jerk, steady throttle. Risk: LOW.

● AGGRESSIVE
High throttle bursts, hard braking, sharp steering. Risk: HIGH.

● FATIGUED
Erratic inputs, micro-corrections, late reactions. Risk: MEDIUM-HIGH.

● SPORTY
High-performance, elevated RPM but controlled. Risk: MEDIUM.

● HIGHWAY CRUISE
Sustained high-speed, low-variance motorway driving. Risk: LOW.
